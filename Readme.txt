Lucy - Chatbot Demo

Lucy is a text-based chatbot that can perform various tasks, including general chat, opening websites, searching the web, providing jokes, playing games like Tic Tac Toe and Hangman, and more. It is too basic with limited features, we look forward to completing the project in the future to be able to help the user in various fields.

Prerequisites:
To run this project, you need to have the following:
1) Java Development Kit (JDK) installed on your system, we have used eclipse Version: 2021-12 (4.22.0)
2) A Java IDE or a text editor to edit and run the Java code

The fundamental things utilized in the program were :

1) Loops 
2) If-else explanation 
3) Run order 
4) Switch explanations
